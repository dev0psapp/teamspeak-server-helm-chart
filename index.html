TeamSpeak Server Helm Chart Repository

This page serves as the base URL for the TeamSpeak Server Helm Chart repository.

To add this repository to your Helm client, use the following command:

helm repo add teamspeak https://dev0psapp.github.io/teamspeak-server-helm-chart/

You can then search for and install charts:

helm repo update
helm search repo teamspeak
helm install my-teamspeak teamspeak/teamspeak

For more details, you can view the raw Helm repository index file or download the TeamSpeak chart package.

The source code for this chart is available on GitHub.
